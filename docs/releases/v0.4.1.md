## highlights

- Added a minimal library target for the root `majutsu` package so docs.rs can
  build package documentation with `cargo rustdoc --lib`.
- Improved `scripts/publish-crates-io.sh` to skip package versions that already
  exist on crates.io, reducing unnecessary registry requests on reruns.
- Documented the `jq` dependency and skip behavior in the crates.io release
  guide.
