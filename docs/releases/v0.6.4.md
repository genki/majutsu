# majutsu v0.6.4 release evidence

## release

- version: 0.6.4
- date: 2026-06-29 JST
- version output: `mj 0.6.4`

## summary

- Fixed a `mj state` pager regression. TTY output that is still being produced
  after the initial prefetch window now enters the interactive viewer instead
  of falling back to direct output.
- Preserved the fast path for short completed `mj state` output: when the
  result is complete and fits within the terminal height, it prints directly.
- Added a unit test for the prefetch decision and a Linux PTY regression test
  using `script -qec` to verify that tall `mj state` output enters the
  alternate-screen viewer.
