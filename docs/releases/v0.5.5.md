# majutsu v0.5.5 release evidence

## release

- version: 0.5.5
- date: 2026-06-22 JST
- version output: `mj 0.5.5`

## summary

- Fixed `mj log --root <root> --full` so operation records that reference
  pruned local snapshot metadata no longer abort the whole log with
  `Query returned no rows`.
- Added regression coverage for root-filtered full log output after snapshot
  metadata has been pruned.
- Kept `mj log --operations` as the internal operation-record view; the normal
  change log now skips operations whose before/after snapshot metadata is no
  longer available for file-level diff rendering.
