# majutsu v0.5.9 release evidence

## release

- version: 0.5.9
- date: 2026-06-24 JST
- version output: `mj 0.5.9`

## summary

- Added `mj note` for showing and editing human notes on existing operations.
- Allowed `mj note` to accept `snap-...` references by resolving them to the
  operation that created the snapshot.
- Rewrote the local operation log after note edits so DB state and oplog output
  remain aligned.
- Documented note workflows in README, command guide, CLI layout, and
  operations runbook.
