# majutsu v0.5.7 release evidence

## release

- version: 0.5.7
- date: 2026-06-22 JST
- version output: `mj 0.5.7`

## summary

- Hardened snapshot reads for symlink-swap / TOCTOU resistance when roots do
  not follow symlinks.
- Added Linux `openat2(RESOLVE_BENEATH | RESOLVE_NO_SYMLINKS)` snapshot reads.
- Added Unix/macOS `openat + O_NOFOLLOW` snapshot reads.
- Added Windows reparse-point rejection plus opened-handle final-path boundary
  checks for snapshot reads.
- Hardened restore and materialized mount path-boundary checks for symlink
  write/delete escapes.
- Quarantined executable remote hook/plugin config during `mj clone` unless
  `--trust-remote-hooks` is explicitly used.
- Added decoded-size limits for zstd-compressed remote metadata/head objects.
- Disabled S3 HTTP redirects and rejected non-local plaintext HTTP endpoints
  unless explicitly allowed for trusted testing.
- Restricted Unix state/key permissions for encrypted homes.
