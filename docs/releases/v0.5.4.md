# majutsu v0.5.4 release evidence

## release

- version: 0.5.4
- date: 2026-06-22 JST
- version output: `mj 0.5.4`

## summary

- Normalized Windows file remote URL forms so `file://C:\...`,
  `file:///C:/...`, and `file:/C:/...` all resolve to the intended local
  Windows path.
- Added cross-platform unit coverage for Windows file URL suffix parsing and
  Windows-only runtime coverage for the accepted file remote URL variants.
- Updated Windows getting-started and platform support docs to show S3-style
  remote setup plus local file remote URL forms.
