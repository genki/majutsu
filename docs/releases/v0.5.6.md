# majutsu v0.5.6 release evidence

## release

- version: 0.5.6
- date: 2026-06-22 JST
- version output: `mj 0.5.6`

## summary

- Added LANG-aware human-facing CLI help descriptions for English, Japanese,
  Chinese, Spanish, and French while keeping machine-readable keys stable.
- Added `mj restore --ago <duration>` for compact relative restore selectors.
- Added `mj restore --op <op-prefix>` and operation-id prefix resolution for
  `mj op show`, `mj op diff`, and `mj op restore`.
- Changed offsetless restore time parsing to use the environment local
  timezone instead of UTC.
- Added a Podman-first clean container demo script and documentation.
- Added restore-focused tutorial documentation.
