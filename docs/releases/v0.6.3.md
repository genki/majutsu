# majutsu v0.6.3 release evidence

## release

- version: 0.6.3
- date: 2026-06-29 JST
- version output: `mj 0.6.3`

## summary

- Aligned `mj state` with the root lifecycle model. With no reference, it now
  treats the basis as infinite past and reports tracked paths from root
  registration onward.
- Added Git-style path filtering for `mj state` and `mj log` with `-- <path>`.
  Root-local output stays root-relative while global output accepts
  `root/path`.
- Added `-U/--untrack` for root-scoped untracked file discovery. Untracked live
  files are shown as `?`, sorted before tracked rows, and untracked directories
  are summarized without recursive expansion.
- Added short options `-D` for `--deleted` and `-d` for `--diff`.
- Improved `mj state -d/--diff` so text changes are rendered as compact,
  colored hunks instead of dumping unrelated file content.
- Kept large and batch-added files out of automatic tracking by default while
  making them visible through `mj state -U --status '?'`.
- Added regression coverage for default state semantics, pathspec filtering,
  untracked rows, deleted filtering, diff shorthand, large/batch auto-track
  policy, status paging decisions, ANSI truncation, and database compact CLI
  reporting.
