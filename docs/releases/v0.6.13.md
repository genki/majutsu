# majutsu v0.6.13 release evidence

## Release

- version: 0.6.13
- date: 2026-07-15 JST
- version output: `mj 0.6.13`

## Summary

- Hardened daemon lifecycle handling against PID reuse, stale runtime files,
  duplicate starts, and unrelated process termination.
- Serialized snapshot, sync, prune, and GC maintenance paths to prevent remote
  objects from being removed while a concurrent sync still needs them.
- Added bounded retry and backoff to transient process, filesystem, database,
  and remote-store failure paths.
- Improved durable event-journal recovery and maintenance scheduling so work
  is deferred rather than overlapped or lost.
- Added macOS RSS measurement for the watch memory guard.
- Switched HTTP transport from native OpenSSL to rustls for reproducible cross
  compilation.
- Added the SSH-controlled machinaai remote build service with bounded sccache,
  disposable Cargo targets, macOS native builds, and Linux cargo-zigbuild.
