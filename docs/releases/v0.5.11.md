# majutsu v0.5.11 release evidence

## release

- version: 0.5.11
- date: 2026-06-24 JST
- version output: `mj 0.5.11`

## summary

- Colored the root-name prefix in `mj log` file paths consistently with
  `mj state`.
- Applied the same root-prefix color treatment to snapshot diff output paths
  such as `mj diff` and `mj op diff`.
- Added regression coverage for colored `mj log` and `mj diff` path output.

## local validation

```sh
cargo fmt --all --check
cargo check --locked
scripts/sync-internal-crates.sh --check
cargo clippy --locked --all-targets -- -D warnings
cargo test --locked --test remote_file log_defaults_to_managed_file_changes_not_sync_operations -- --exact
cargo test --locked --test remote_file diff_reports_added_modified_and_deleted_paths -- --exact
cargo test --locked --workspace
```

- result: pass

## runtime validation

```sh
env -u NO_COLOR MJ_COLOR=always mj log --limit 1
env -u NO_COLOR MJ_COLOR=always mj diff
```

- result: root prefixes in mixed `root/path` output are rendered with the same
  bright root-name color used by `mj state`.
