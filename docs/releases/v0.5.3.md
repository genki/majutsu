# majutsu v0.5.3 release evidence

## release

- version: 0.5.3
- date: 2026-06-21 JST
- version output: `mj 0.5.3`

## summary

- Improved `mj state <ref>` with `-r/--root`, `-j/--json`, `--diff`, and
  `--meta` so Git/Jujutsu users can inspect root-scoped changes with optional
  colored line diffs.
- Hid metadata-only changes from default `mj state` output and exposed them as
  lowercase `m` with `--meta`.
- Improved CLI discoverability while keeping maintenance commands top-level:
  command grouping help, `commit` alias for `snapshot`, `switch` alias for
  `branch switch`, and `restore mount/unmount/hydrate`.
- Restored `mj root size` streaming behavior so local scan results are shown
  before slow remote object listing, and added `--no-remote-cache` for direct
  backend scans.
