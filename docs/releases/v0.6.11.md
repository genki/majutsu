## summary

- Made deep sync status and remote repair protect the current state by default,
  with explicit `--history` checks for retained historical snapshots.
- Removed stale cached remote refs after remote or host-prefix changes.
- Allowed prune to discard non-current history whose remote payload is already
  unavailable, and rewrote pack indexes after historical blob metadata removal.
- Recovered stale queue atomic-write temporary files after a one-hour safety
  window.
- Parallelized independent GCS-compatible prefix LIST and cache HEAD requests.
  Full fsck on the vagrant production state improved from 20.28 seconds to
  5.33 seconds while preserving full local and remote reachability checks.
