# majutsu v0.6.7 release evidence

## release

- version: 0.6.7
- date: 2026-06-30 JST
- version output: `mj 0.6.7`

## summary

- Fixed a `mj state` TTY pager regression where output that exceeded the
  terminal height after the initial prefetch window could still fall back to
  direct streaming.
- Changed `mj state` pager selection to wait until either the terminal-height
  overflow is observed or the producer finishes, preserving the no-pager path
  for short completed output.
- Added a delayed-overflow regression test so slow-producing state output is
  still routed to the viewer when it eventually exceeds the terminal height.
