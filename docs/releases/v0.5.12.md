# majutsu v0.5.12 release evidence

## release

- version: 0.5.12
- date: 2026-06-25 JST
- version output: `mj 0.5.12`

## summary

- Added watch daemon RSS guardrails through `--max-rss-mib` and
  `MAJUTSU_WATCH_MAX_RSS_MIB`.
- The default watch daemon RSS limit is 2048 MiB; `0` disables the guard.
- When the guard trips, the daemon records `watch-memory-limit` and exits
  explicitly instead of continuing toward host-wide OOM.
- Generated systemd services now include `MemoryMax=` and `OOMPolicy=stop`.
- Documented high-churn root guidance and the memory guard path.
