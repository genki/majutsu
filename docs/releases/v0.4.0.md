## highlights

- Added `mj --system` for a root-owned system instance with `/etc/majutsu/config.toml`
  and `/var/lib/majutsu` defaults.
- Added `mj daemon service --scope user|system` so user daemons and root-owned
  systemd system daemons can be rendered separately.
- Improved `mj root list` from raw TSV to a width-aware table with total,
  active, and issue counts.
- Improved `mj daemon status` into grouped `Daemon`, `Queues`, `Journal`,
  `Memory`, and `Root Status` sections.
- Improved `mj root size` by separating client-side columns from remote-side
  backend accounting with `|`, and by showing `backend` and pack-adjusted `used`
  side by side.
- Continued local-only release validation; GitHub Actions are not used as the
  release gate.
